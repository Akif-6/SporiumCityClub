# 🏋️‍♂️ Sporium City Club | Full-Stack Gym Management System

## 🚀 Proje Hakkında
Sporium City Club; modern, karanlık tema (dark mode) odaklı ve yüksek performanslı bir spor salonu yönetim ve tanıtım web projesidir. Frontend tarafında herhangi bir ağır framework kullanılmadan, tamamen saf (vanilla) dillerle maksimum hız hedeflenmiş; arka planda ise güçlü ve güvenli **C# .NET 8.0 API** mimarisi ile donatılmıştır. 

*"Kalabalıktan uzak, hedef odaklı spor deneyimini"* dijital dünyada en şık haliyle sunmak için tasarlandı!

## 🛠️ Kullanılan Teknolojiler (Tech Stack)
* **Frontend (Önyüz):** HTML5, CSS3, JavaScript (Vanilla)
* **Backend (Arkayüz):** C# .NET 8.0 Web API
* **Veritabanı:** MySQL
* **Mimari:** API / Mikroservis Yaklaşımı (Bağımsız Ön ve Arka Yüz)

## ✨ Öne Çıkan Özellikler
* **Premium Tasarım:** Neon yeşil detaylarla (Zehir Yeşili) bezenmiş, her ekrana %100 uyumlu (Responsive) karanlık tema.
* **Akıcı Animasyonlar:** Efsanevi giriş ekranı (Pulse efektli Preloader), yumuşak sayfa geçişleri ve sayfayı kaydırdıkça beliren elementler (Intersection Observer).
* **Üye Paneli:** Üyelerin antrenman programlarını, beslenme planlarını ve kişisel gelişimlerini takip edebildiği gelişmiş dashboard.
* **Yönetim (Admin) Komuta Merkezi:** Salon yöneticisinin tüm sistemi (üyeler, antrenörler, ödemeler) tek bir ekrandan yönetebildiği profesyonel admin paneli.
* **Clean Code & Mimari:** Temiz URL (Clean URL) yapısı, optimize edilmiş kodlama ve kolay genişletilebilir altyapı.

## ⚙️ Kurulum ve Çalıştırma

Bu proje profesyonel standartlara uygun olarak iki bağımsız parçadan (Frontend ve Backend API) oluşmaktadır.

### 1. Frontend (Önyüz) Çalıştırma
Herhangi bir ekstra derleyiciye ihtiyaç duymaz. 
1. Proje dosyalarını indirin.
2. Klasör içindeki `index.html` dosyasına çift tıklayarak tarayıcıda açın veya VS Code üzerinden *Live Server* eklentisi ile çalıştırın.

### 2. Backend (API) Çalıştırma
1. C# API klasöründeki `.sln` (Solution) dosyasını **Visual Studio 2022** ile açın.
2. Bilgisayarınızda **.NET 8.0 SDK**'sının kurulu olduğundan emin olun.
3. Projeyi derleyip (Build) yukarıdaki yeşil `Play` butonuna basarak API sunucusunu (localhost) ayağa kaldırın. (Swagger arayüzü sizi karşılayacaktır).
